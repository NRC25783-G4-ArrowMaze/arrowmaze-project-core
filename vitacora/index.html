Arrow Maze · Project Core

La fuente única de verdad de un puzzle game hecho spec primero

Arrow Maze es un puzzle de flechas sobre un tablero-grafo, diseñado con Clean Architecture + DDD y construido con Specification-Driven Development: cada decisión se discute, se especifica en Gherkin y se registra antes de escribir código. Aquí puedes ver cómo se hizo — y la bitácora completa del uso de IA.
